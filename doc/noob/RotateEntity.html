RotateEntity

 Rotates an entity in the given scene by the given yaw, pitch and roll.

Syntax

 void RotateEntity(
  std::string entityId,
  float yaw,
  float pitch,
  float roll
 );


Parameters

 std::string entityId
  The id of the entity to be rotated.
 float yaw
  The amount of yaw to add.
 float pitch
  The amount of pitch to add.
 float roll
  The amount of roll to add.

Return value

 This function returns void.
