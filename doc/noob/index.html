Noob

The noob is a simple user interface designed to let people with minimal programming
experience create games using the engineD game engine.

When creating the noob interface you must supply it with a window title for your game.
The noob has the following main functions:
  • GameLogic
  • LoadSceneFile
  • MoveEntity
  • RotateEntity
  • Run
