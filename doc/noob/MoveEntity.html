MoveEntity

 Moves an entity by the given x, y and z.

Syntax

 void MoveEntity(
  std::string entityId,
  float x,
  float y,
  float z
 );


Parameters

 std::string entityId
  The id of the entity to be moved.
 float x
  The amount to move on the x axis.
 float y
  The amount to move on the y axis.
 float z
  The amount to move on the z axis.

Return value

 This function returns void.
