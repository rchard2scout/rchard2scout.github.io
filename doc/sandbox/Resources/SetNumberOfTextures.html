SetNumberOfTextures

 Sets the number of textures for a mesh with the given id.

Syntax

 void SetNumberOfTextures(
  std::string id
  DWORD numberOfTextures
 );


Parameters

 std::string id
  The key of the resource.
 DWORD numberOfTextures
  The number of textures.

Return value

 This function returns void.
