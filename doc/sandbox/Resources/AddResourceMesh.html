AddResourceMesh

 Creates a new mesh and adds it to the resources managed by the resource manager.

Syntax

 bool AddResourceMesh(
  std::string id,
  std::string fileName,
  std::string fileType,
  Renderer* renderer
 );


Parameters

 std::string id
  A key for the new resource mesh.
 std::string fileName
  The file location and name used to create the mesh.
 std::string fileType
  The type of file to be used to create the mesh.
 Renderer* renderer
  A pointer to the renderer to be used to create the new mesh.

Return value

 This function returns true if the mesh is created succesfully.
