AddResourceLoader

 Adds the given resource loader to the resource manager with the given key.

Syntax

 void AddResourceLoader(
  std::string id,
  ResourceLoader* loader
 );


Parameters

 std::string id
  A key for the new resource loader.
 ResourceLoader* loader
  A pointer to the new resource loader.

Return value

 This function returns void.
