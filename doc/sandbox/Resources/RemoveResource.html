RemoveResource

 Removes the resource, corresponding with the given key, from the resource manager.

Syntax

 bool RemoveResource(
  std::string id
 );


Parameters

 std::string id
  The key of the resource.

Return value

 This function returns true if the removal of the resource was succesful.
