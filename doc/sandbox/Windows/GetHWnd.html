GetHWnd

 Returns the hWnd key of the window with the given id key.

Syntax

 HWND GetHWnd(
  std::string id
 );


Parameters

 std::string id
  The id key of the window.

Return value

 This function returns the hWnd key of the window corresponding to the id key.
