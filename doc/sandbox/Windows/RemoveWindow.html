RemoveWindow(id)

 Removes the window with the given id key.

Syntax

 bool RemoveWindow(
  std::string id
 );

 bool RemoveWindow(
  HWND hWnd
 );


Parameters

 std::string id
  The id key of the window.

 HWND hWnd
  The hWnd key of the window.

Return value

 This function returns true if the window corresponding to the
 id or hWnd key was removed succesfully.
