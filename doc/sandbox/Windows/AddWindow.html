AddWindow

 Creates a new window and adds it to the windows managed by the window manager.

Syntax

 Window* AddWindow(
  std::string id,
  std::string windowName,
  int x,
  int y,
  int width,
  int height
 );


Parameters

 std::string id
  A key for the new window.
 std::string windowName
  The title that will be visible above the window.
 int x
  The amount of pixels the window will be from the left screen border.
 int y
  The amount of pixels the window will be from the top screen border.
 int width
  The width of the window in pixels.
 int height
  The height of the window in pixels.

Return value

 This function returns a pointer to the newly created window.
