RemoveInput

 Removes the input device, corresponding to the given id, from the input managers map.

Syntax

 bool RemoveInput(
  std::string id
 );


Parameters

 std::string id
  The string key corresponding to the input device to be removed.

Return value

 Function returns true if removal of the input device from the map was succesful.
