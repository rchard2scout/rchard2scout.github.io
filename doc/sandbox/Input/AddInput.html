AddInput

 Adds an input device to the InputManager's map using the given id as the key.

Syntax

 bool AddInput(
  std::string id,
  Input* input
 );


Parameters

 std::string id
  A string id to be used as the key.
 Input* input
  A pointer to the input device to be saved in the map

Return value

 Returns true if the input device is placed succesfully.
