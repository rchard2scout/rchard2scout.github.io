TranslateWorld

 Translates the world of the given scene by the given x, y and z relative to the given camera.

Syntax

 void MoveEntity(
  std::string sceneId,
  float dx,
  float dy,
  float dz,
  std::string cameraId
 );

 void MoveEntity(
  Scene* scene,
  float dx,
  float dy,
  float dz,
  std::string cameraId
 );


Parameters

 std::string sceneId
  The id of the scene in which the world exists.
 Scene* scene
  The pointer to the scene in which the world exists.
 float dx
  The amount to translate on the x axis.
 float dy
  The amount to translate on the y axis.
 float dz
  The amount to translate on the z axis.
 std::string cameraId
  The id of the camera.

Return value

 This function returns void.
