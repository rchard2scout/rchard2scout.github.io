AddCamera

 Adds a camera to the scene corresponding with the given sceneId.
 The camera renders to the given window.

Syntax

 bool AddCamera(
  std::string sceneId,
  std::string id,
  Window* window
 );


Parameters

 std::string sceneId
  The id of the scene to which the camera will be added.
 std::string id
  A key for the new camera.
 Window* window
  A pointer to the window to which the new camera will render.

Return value

 This function returns true if the camera has been added succesfully.
