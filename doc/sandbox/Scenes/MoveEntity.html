MoveEntity

 Translates an entity in the given scene by the given x, y and z.

Syntax

 void MoveEntity(
  std::string sceneId,
  std::string entityId,
  float x,
  float y,
  float z
 );


Parameters

 std::string sceneId
  The id of the scene in which the entity exists.
 std::string entityId
  The id of the entity to be moved.
 float x
  The amount to translate on the x axis.
 float y
  The amount to translate on the y axis.
 float z
  The amount to translate on the z axis.

Return value

 This function returns void.
