InitTerrain

 Initializes the terrain of the given scene using the given renderer.
 Load the height data from the heightmap into a vertexbuffer and calculates texture coordinates.

Syntax

 bool InitTerrain(
  std::string sceneId
  Renderer* renderer
 );


Parameters

 std::string sceneId
  The id of the scene in which the terrain exist.
 Renderer* renderer
  A pointer to the renderer to be used to initialize the terrain.

Return value

 This function returns true if the terrain is initialized succesfully.
