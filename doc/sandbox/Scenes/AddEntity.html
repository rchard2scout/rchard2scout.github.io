AddEntity

 Adds an entity to the scene corresponding with the given sceneId.
 The entity will be placed at the given location with the given rotation.

Syntax

 bool AddEntity(
  std::string sceneId,
  float x,
  float y,
  float z,
  float yaw,
  float pitch,
  float roll,
  std::string id
 );


Parameters

 std::string sceneId
  The id of the scene to which the entity will be added.
 float x
  The x location of the new entity.
 float y
  The y location of the new entity.
 float z
  The z location of the new entity.
 float yaw
  The yaw rotation of the new entity.
 float pitch
  The pitch rotation of the new entity.
 float roll
  The roll rotation of the new entity.
 std::string id
  A key for the new entity.

Return value

 This function returns true if the entity has been added succesfully.
