ChangeRenderer

 Changes the currently used renderer into the given one.

Syntax

 void ChangeRenderer(
  HWND hWnd,
  Renderer* renderer
 );


Parameters

 HWND hWnd
  A key to the window to which the renderer should render.
 Renderer* renderer
  A pointer to the new renderer.

Return value

 This function returns void.
